How It Works

When the page loads, Emend.js:

  1. Searches for all <a> elements with mailto: links
  2. Encrypts the email addresses using the provided salt
  3. Stores the encrypted value in a data-emended-mailto attribute
  4. Changes the href attribute to # to prevent scraping
  5. Attaches click handlers to restore the functionality when users click

This makes the email addresses invisible to spam bots while preserving the user experience.

Code Example


// Import the library
import emend from '/path/to/emend.esm.js';

// Initialize when DOM is loaded
document.addEventListener('DOMContentLoaded', () => {
  emend.init({
    salt: 'MySecretSalt'
  });
});
