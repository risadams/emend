Emend.js

A lightweight JavaScript library to protect email addresses from web scrapers

Current version: loading...

Overview

Emend.js is a modern JavaScript library that protects mailto: links in your HTML documents from being harvested by spam bots and web scrapers, while maintaining functionality for human users.

How It Works: Emend.js encrypts email addresses and stores them as data attributes, replacing the actual mailto: links with harmless anchors. When a user clicks the link, the email is decrypted on-the-fly and the mail client is opened normally.

Installation

NPM

npm install emend

Import as ES Module

import emend from 'emend';

Use with Script Tag

<script src="/path/to/emend.js"></script>

Basic Usage

// Initialize the library
document.addEventListener('DOMContentLoaded', () => {
  emend.init({
    salt: 'YourSecretKey'  // Use a unique salt for better security
  });
});

Configuration Options

Option Default Description
salt "" Secret key used for encoding/decoding emails
explicitPrefix "@" Prefix for explicitly marked links
explicitOnly false When true, only protect links with the explicit prefix
sendClickDelay 500 Milliseconds to wait before triggering click events
domRemoveDelay 1200 Milliseconds to wait before removing temporary DOM elements

API Reference

emend.init(options)

Initializes the library with optional configuration. Returns the emend instance for chaining.

emend.encode(email)

Manually encode an email address or string. Returns the encoded string.

emend.decode(encodedValue)

Decode a previously encoded string. Returns the original email address.

emend.protect(element)

Manually protect a specific anchor element. Returns the element for chaining.

emend.sendMail(element)

Manually trigger the email client for a protected link.

emend.version

Get the current version of the library.
